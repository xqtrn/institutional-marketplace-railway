Overview

DriveNets is a pioneer in cloud-native networking software that enables large-scale network modernization and AI infrastructure. Founded in 2015 in Ra'anana, Israel by Ido Susan and Hillel Kobrinsky, DriveNets developed Network Cloudâ€”a software-based, cloud-native routing system built on a shared infrastructure of standard networking white boxes. The platform disaggregates network hardware from software, dramatically reducing costs while increasing agility. DriveNets Network Cloud is deployed as the core routing solution for AT&T, one of the largest networks in the world, demonstrating the platform's ability to handle massive scale and complexity.
