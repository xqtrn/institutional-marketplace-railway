Overview

Dremio is the SQL lakehouse platform that enables enterprises to run sub-second analytics directly on data stored in cloud data lakes like Amazon S3, Azure Data Lake, and Google Cloud Storage. Founded in 2015 in Santa Clara, California, Dremio eliminates the need for data copying and movement by providing a high-performance query engine that sits on top of existing data lake infrastructure. The platform supports Apache Iceberg and other open table formats, enabling organizations to build modern data architectures without vendor lock-in. Dremio is used by leading enterprises to power business intelligence dashboards, data science workflows, and self-service analytics.
