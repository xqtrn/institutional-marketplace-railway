Overview

DFINITY Foundation is a Swiss non-profit developing the Internet Computer Protocol (ICP), a blockchain that extends the functionality of the public internet to host smart contracts, web applications, and data at scale. Founded by computer scientist Dominic Williams, the project aims to create a 'World Computer' that allows developers to build and run web applications entirely on blockchain without traditional cloud providers. The ICP network features 500% year-on-year growth in on-chain compute, AI capabilities for smart contracts, and cross-chain interoperability with Bitcoin and Ethereum. The ecosystem has distributed over $6M in grants to 239 projects across 41 countries.
